OMAP PSP Driver WebEx Presentations[edit]

  • TI is conducting a series of Webexes on OMAP PSP Drivers.
  • These Webexes can be accessed via the table below. Note that the wrf has the Webex Player embedded in the stream - you don't need to download anything else.
  • The data in the WebEx Recordings correspond to the OMAP 35x Linux PSP 1.0.0 release available in the PSP Update advisor site at :- https://www-a.ti.com/downloads/sds_support/targetcontent/psp/omap35x/index.html
Sl.No OMAP Web-Ex Recording Description Remarks
1 OMAP Video : Session 1 : Linux Video Device Driver Framework Link Users would understand about :-
V4L2 Framework, FBDEV Framework and SYSFS Interface
Framework provided by the Linux for developing the device drivers for video hardware. This is important for understanding and using the driver provided features.
About tuning the driver for specific customer requirements.
2 OMAP Video : Session 2 : OMAP video driver Implementation Users would understand about :-
The different feature set provided by the drivers and its usage
Tuning the driver for specific customer requirements.
Future road map in terms of new features
Driver limitations
Display Driver, Resizer Driver and Capture Driver 		The information in slides 31 and 32 regarding daughter card for the engineering development team. TI plans to have a similar daughter card for Video around 4q 08 for customers
3 OMAP Power Management : Under the hood Users would understand about :-
Key Hardware Features
Summary of Software Features Frameworks (including call flow thru sources):
- CPU Idle
- CPU Freq
Different power management Frameworks supported in the OMAP35x PSP |
4 OMAP Linux Device Driver Performance Test Bench Linux Performance Test Bench (LPTB) Overview
Performance measurement (Throughput and CPU load)
Data sheet walkthrough
5 OMAP MMC/SD/SDIO/CE-ATA Driver Users would learn about :-
Overview of MMC/SD/SDIO/CE-ATA Linux MMC/SD stack
Usage of drivers
Various storage technologies and their realization under Linux
6 OMAP NAND Driver Users would learn about :-
Overview of NAND/OneNAND
Driver Architecture, Features and Application Interface
Various storage technologies and their realization under Linux
7 OMAP Storage Driver RoadMap Users would learn about Storage Driver Roadmap for OMAP35x PSP
8 OMAP Audio Driver Users would understand about :-
ALSA framework
Linux audio driver Architecture and configuration
Key hardware and software features
Driver limitations
Environment setup for ALSA Aplay/Arecord
Future: ALSA
Usage of common ALSA applications (aplay, arecord etc)
Future road map (ALSA SOC) 		The Daughter card mentioned in slide 41 was for the Engineering team only for testing out the driver and is not available externally
9 OMAP USB: Protocol and Driver Users would understand about :-
Signaling details (hopefully just right enough)
OTG core overview
Linux USB stack.
Protocol and how the Linux USB stack works.
Will know about hotplugging support on Linux and a class driver overview
