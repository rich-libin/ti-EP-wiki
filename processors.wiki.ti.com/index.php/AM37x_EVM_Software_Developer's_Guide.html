Welcome to the Sitara Linux Software Developer's Guide[edit]

Thank you for choosing to evaluate one of our Sitara ARM microprocessors. Please bookmark this page and refer back to it as needed. It is designed to quickly provide the information you need most while evaluating the AMx microprocessor, specifically running one of the Software Architectures available, embedded Linux. We are always striving to improve this product.  Please let us know if you have ideas or suggestions.  

Version[edit]

This information covers SDK 8.0, which was released in March of 2014. If you are using a different version of the SDK, please refer to the archived SDG for your specific release.

Note: For documentation on Processor SDK please refer to the Processor SDK Linux Software Developer’s Guide.
