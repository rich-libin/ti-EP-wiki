AM437x[edit]

GEL Files[edit]

File:AM437x GELs.zip

This GEL file package for AM437x GP, IDK and Starter Kit EVM includes the following files:

  • AM437x_MMR.gel: Collection of defines for Memory Mapped Registers. Not an exhaustive list, but includes all registers used by other files
  • AM437x_Status: Various functions to decode status registers, enabled CLKOUT signals, and reading PLL configurations
  • AM43xx_PLLconfig: Functions to configure the device PLLs. Only OPP100 is configured since changing OPPs requires voltage changes (voltage changes not supported in GELs)
  • AM43xx_EMIFconfig_HWlvl.gel: Functions to configure DDR3 or LPDDR2 memories. Configurations are specific to EVMs. Certain values will need to change for custom boards.
  • AM437x_EVMs.gel: Modify this to create a new board if needed.
  • evmam437x.gel: AM437x GP EVM Top level GEL which calls the other files.
  • idk_am437x.gel: AM437x IDK EVM Top level GEL which calls the other files.
  • sk_am437x.gel: AM437x Starter Kit EVM Top level GEL which calls the other files.

Currently, these GELs support GP EVM v1.5A, IDK EVM v1.3A and Starter kit EVM v1.1A
