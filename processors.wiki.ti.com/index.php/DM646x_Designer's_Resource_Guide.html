Getting started from DM6467/T EVM & DVSDK[edit]

EVM:

  • DM6467 EVM: You can find related link here.
  • DM6467T EVM: You can find related link here.


FAQ:

What's in the EVM link?

  • EVM schematics, gerber, technical reference guides, CCS test cases and so on.


DVSDK:

  • Latest DVSDK
    • DVSDK 3.1: You can get it here. DVSDK 3.1 is based on open source GIT kernel 2.6.32 and using codesoucery tools.
  • Other DVSDK
    • DVSDK 2.0 : You can get it here. DVSDK 2.00 is based on MV Pr05.0 and MV 2.6.18 Kernel.
    • DVSDK 1.4 : you can download it here using your my.ti.com account. DVSDK 1.40 is based on MV Pr04.01 and MV 2.6.10 Kernel.


FAQ:
What's in DVSDK?

  • encode demo, decode demo, encodedecode demo and so on.

What you can find in DVSDK download link:

  • DVSDK, LSP(Linux support package), U-boot, flash utility, Linux compiling tool (or download link) and so on.

H/W designer's resource[edit]

  1. Refer to the DM6467 EVM H/W design or here for DM6467T.
  2. Go through DM6467 silicon errata or DM6467T silicon errata. In errata, you can also find the part number difference based on different silicon version.
  3. Know the power consumption of DM6467 or here for DM6467T.
  4. Schematic and PCB design:
    • A. High Speed DSP Systems Design Reference Guide.
    • B. Understanding TI's PCB Routing Rule-Based DDR Timing Specification.
    • C. DM6467 datasheet - 7.10.2 DDR2 Interface to implementing DDR2 schematic and PCB Layout on DM6467 or DM6467 datasheet - 7.10 DDR2 Memory Controller for DM6467T.
    • D. You can download the simulation models from the product folders DM6467 or DM6467T
    • E. USB 2.0 Board Design and Layout Guidelines.
  5. TMS320C6000 DSP Designing for JTAG Emulation Reference Guide.
  6. Hardware Design Checklist.
  7. For peripheral timing info (max, min), pls check DM6467 datasheet or DM6467T datasheet.
  8. Desgin boot mode, pls refer to doc and reference code.
  9. Referece Power desgin:
    • A. PIP Power Reference Design for the DM6467
    • B. 5V/12V Input DM6467 (x1) Power Ref Design Using a DC/DC Controller & Converter
    • C. 5V/12V Input DM6467 (x2) Power Ref Design Using a DCDC Controller & Converter
    • D. 5V/12V input DM6467 (x1) Power Reference Design
    • E. 5V/12V input DM6467 (x2) Power Reference Design
  10. DM646x Complementary Products

Driver/LSP designer's resource[edit]

  1. Go through DM6467 silicon errata and know the silicon bug & the workaround.
  2. Download the latest LSP and its datasheet by your my.ti.com account. Please note there are two LSP versions which supports MV2.6.10 kernel and MV2.6.18 kernel separately.
  3. The utilities to boot and flash various TI DaVinci processors.
  4. DM6467 PSP WebEx Presentations
  5. DM6467 Pin Mux Utility.

DaVinci Linux resource[edit]

  1. DaVinci Linux Open Source
  2. Linux articles on this wiki
  3. Download the LSPvia your my.ti.com account. Please note that there are two LSP versions which supports MV2.6.10 kernel and MV2.6.18 kernel separately.
  4. Chinese customers can visit the TI column of China Linux Forum.
