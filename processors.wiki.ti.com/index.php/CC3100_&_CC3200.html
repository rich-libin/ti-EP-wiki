Welcome to the SimpleLink™ Wi-Fi® CC3100/CC3200 Main Page.
[edit]

The objective of this wiki is to provide extensive self-help resources including technical collateral and information for CC3100/CC3200 Wi-Fi® SimpleLink customers. Created for the Internet of Things, the SimpleLink™ Wi-Fi® family dramatically simplifies the implementation of Internet connectivity by integrating & embedding all the protocols for Wi-Fi, Internet and security on the chip. It is a complete platform solution including various tools and software, sample applications, user guides, programming guides, reference designs and the TI E2E™ Community for support. The SimpleLink Wi-Fi family consists of:
  • CC3100 - A Wireless Network Processor with on-chip Wi-Fi, internet, and robust security protocols. It can be used to connect any low-cost, low-power microcontroller (MCU) to the Internet of Things (IoT)!
  • CC3200 - A Wireless MCU integrating high-performance ARM® Cortex™-M4 MCU with on-chip Wi-Fi, internet, and robust security protocols. It can be used to develop an entire IoT application with a single IC.
