Installing JTAG Header on Industrial Communications Engine (ICE)

From Texas Instruments Wiki
Jump to: navigation, search

The Industrial Communications Engine (ICE) development board is deployed with an XDS100v2 JTAG emulator using an FTDI device. This allows to load code through JTAG using CCS5, but only at a limited speed. Some developer may want to increase JTAG speed through use of an external JTAG emulator (XDS510, XDS560, ...). The following article explains how to modify the ICE board in order populate the external JTAG header.

Note: This deactivates the JTAG usage through the FTDI device. However, the UART-to-USB over FTDI device is still possible.

Modification instructions for TMDXICE3359 (ICEv1):

  1. . Get the Samtec FTR-110-03-G-D-06 header, for example at Digikey http://search.digikey.com/scripts/DkSearch/dksus.dll?x=0&y=0&lang=en&site=us&KeyWords=FTR-110-03-G-D-06
  2. . It is good to get ICE schematic and Layout information as reference. This can be found on the ICE product website at http://www.ti.com/tool/tmdxice3359
  3. . Solder the Samtec JTAG header to component position P2. The Samtec header has already pin 6 removed, pay attention to mount it in the right position.
  4. . Remove the following resistors: R239, R240, R241, R242, R243, R244, R245, R246
  5. . Remove R256 (this one is just next to the label P2 on the bottom side)
  6. . Populate the following resistors (0 Ohm): R190, R191, R192, R193, R194, R195, R196

After this change ICE can be used with external JTAG header.


Modification instructions for TMDSICE3359 (ICEv2):

  1. . Get the Samtec FTR-110-03-G-D-06 header, for example at Digikey http://search.digikey.com/scripts/DkSearch/dksus.dll?x=0&y=0&lang=en&site=us&KeyWords=FTR-110-03-G-D-06
  2. . It is good to get ICE schematic and Layout information as reference. This can be found on the ICE product website at http://www.ti.com/tool/tmdsice3359
  3. . Solder the Samtec JTAG header to component position J15. The Samtec header has already pin 6 removed, pay attention to mount it in the right position.
  4. . Remove the following resistors: R142, R190, R192, R194, R201, R202, R203, R204, R209
  5. . Populate the following resistors (0 Ohm): R17, R20, R22, R23

After this change ICE can be used with external JTAG header.
