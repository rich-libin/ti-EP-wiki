Overview[edit]

This category "Code Composer Studio v3" covers the 3.0, 3.1, 3.2 and 3.3 releases.  There is a newer Code Composer Studio v4 release that is recommended for new projects however CCSv3.3 is still very widely used.  Code Composer Studio v3.x included the IDE, debugger, TI emulation drivers, simulators, compilers and DSP/BIOS.  Emulation drivers for 3rd party emulators need to be downloaded separately from the emulator vendor.  Note that CCSv3.3 is the last release to support the F24x/24xx family of microcontrollers.
