[资源简介] About this resource guide[edit]

There are plenty of resource in TI website and here just make it clearer for you to get the entrance rapidly without much documentation first. You can go thru these resource link and get the concept for the chip / platform quickly.

For deeply development, you should read the specs, Application notes and resource in AM37x WiKi carefully. For any question / issue, you can leave your message to TI E2E™ Community - Sitara™ ARM® Microprocessors.

If you want to see a Chinese AM/DM37x Resource Guide,please go to AM/DM37x_Resource_Guide_(Chinese_revision)

[硬件设计资源] H/W designer's resource[edit]

TI provides a rich set of documents for a good hardware design.

  1. [评估板]TI provides the verified reference boards for your development that is EVM. For AM/DM37x, the EVM is from Mistral.
Refer to AM/DM37x EVM H/W design, you need to register Mistral for further downloading.
  1. [芯片资料]Go to AM37x chip documents DM37x chip documents for the latest chip User Guides(TRM), Data sheet (DM), and Silicon Errata(Silicon bug and the workaround).
  2. [原理图设计]You can get the schematic check list for your reference.
  3. [PCB设计]PCB design:
    • AM/DM37x Via Channel Array PCB layout
    • USB 2.0 Board Design and Layout Guidelines
    • Refer to appendix C of nFBGA Packaging application note for thermal modeling results
    • [OrCAD, Allegro封装]You can find them in AM37x OrCAD, Allegro and DM37x OrCAD, Allegro .
    • [BSDL,IBIS仿真模型]You can get the Simulation Models for AM/DM37x here.
    • [焊接温度曲线]Package Reflow Profiles could be found here.
  4. [电源设计]Power design:
    • On the EVM board, TPS65950 is used as the power supplier. Because TPS65950 is 0.4mm pitch package, which is hard for manufacture, TPS65930 0.65mm pitch or TPS65920, with reduced pin out & function, could be used.
    • Besides TPS659xx, TPS65023 and TPS65073 could also be used with AM/DM37x.
    • AM/DM37x with TPS65023: Design In Guide
    • AM/DM37x with TPS65073: Design In Guide
  5. [AM/DM37x功耗评估工具]AM/DM37x Power Estimation Spreadsheet
  6. [OMAP35x到AM/DM37x的硬件移植参考]If you are familar with TI , it's easy to migrate to AM/DM37x. Please find the migration guide here.
  7. [无线连接功能]ARM Processor Wireless Connectivity

[软件设计资源] S/W(including driver) designer's resource[edit]

TI has rich reference software for customer development including Linux, Android, WINCE and ITBOK.

  • [Linux]TI has developed Linux PSP 03.00.xx.xx for AM/DM37x(PSP means Platform Support Products). With each PSP release, there are Release Notes, UserGuide, and Feature Performance Guide. Besides these, there are plenty of application notes, such as: how to setup the environment, toolchain setup, etc. Please find all these from Sitara Linux.
Please download the newest Linux PSP SDK source code in below website:
Linux for AM37X downloads
Linux for DM37X downloads
You can find more info like Graphics SDK from Graphics_SDK_Downloads.
* Suggest use ubuntu 10.04 version for AM/DM37x SDK4.0 version
  • [Android]Based on PSP 03.00.xx.xx for AM/DM37x, there is TI_Android_GingerBread_2_3_DevKit_1_0 and Android-Devkit V02_02_00. It's the snapshot of TI rowboat project, with full kerenel source code, testing, release notes and userguide.
  • [Windows CE]For OMAP35x, AM35x and AM37x, You can find WinCE 6.0 SDK and WinCE 7.0 SDK in here. 
  • [硬件诊断程序]The ITBOK is a program, which is focusing on the Board components verification. For AM/DM37x EVM, please find at ITBOK.
  • [OMAP35x到AM/DM37x的软件移植指导]If you are familar with OMAP35x, it's easy to migrate to AM/DM37x. Please find the SW migration guide OMAP35x_To_AM37x_Software_Migration_Guide.

[工具链] Tools & Utilities[edit]

Maybe some additional utilities are needed for downloading, compiling, and development.

  • [Linux及Android工具链] First to download ubuntu here : Ubuntu SW.
For more detail info, please refer to the below website:Installing the Toolchain
  • [WINCE工具链]The following software must be installed prior to installing the AM35x BSP. Note that the software must be installed in the order listed:
    • Visual Studio 2005 (VS2005)
    • Visual Studio SP1 (Service Pack 1)
    • Windows Embedded CE 6.0 (Plug In for Visual Studio 2005)
    • Windows CE 6.0 SP1 (Service Pack 1)
    • Windows CE 6.0 R2 Update
    • Windows CE 6.0 Cumulative Product Update Rollup Package 2007 (QFEs)
    • Remaining Windows CE 6.0 QFEs
    • You can download most of these tools except Visual Studio 2005(could not be downloaded online) at below website: TOOL Download
  • [ROM CODE下载工具]You can download the Flash Loading Utility for AM/DM37x here
