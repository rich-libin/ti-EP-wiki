Flash Emulation Tool (FET) Overview[edit]

For downloading the sample demo to the EVM, you require MSP430 USB Debugging Interface. This needs to be purchased separately.

NOTE: This is not required if you are using EZ430-RF256x. But jut keep in mind that is is Not Recommended for New Designs

Audio sink board MSP430F5438A EZ430-RF256x
Supported by CCS YES YES YES
Supported by IAR YES YES YES
FETPRO430 YES YES

You need to connect the FET USB Debugging Interface to JTAG debugging port(J2) and you need to connect the USB of FET USB Debugging Interface to the PC.

And the detailed flashing procedure is mentioned here
