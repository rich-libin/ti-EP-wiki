AM3517 One-day Workshop[edit]

This page contains all the materials necessary to present or take a self-paced review of Texas Instruments' AM3517 One-day Workshop (as developed by TI's Technical Training Organization.) For more information on live and self-paced training's available from Texas Instruments please visit the training webpage at

http://focus.ti.com/general/docs/traininghome.tsp

Recorded Lab and Lecture[edit]

To view recordings of the lecture sections as well as a lab walkthrough, use the following link:

http://software-dl.ti.com/trainingTTO/trainingTTO_public_sw/am3517_videos/am3517.html

Note that you may need to enable active-X controls in your web browser. You will also need Windows Media Player version 7 or later installed on your computer.

Hardware Requirements[edit]

Required Hardware[edit]

  • Laptop PC:
    • Must have at least 40 GB harddrive
    • Windows XP operating system installed
    • 1 RJ-45 Ethernet port
    • 1 USB (A-connector) port
    • Power supply
  • Logic PD Zoom AM3517 Experimenter Kit :
    • http://logicpd.com/products/development-kits/zoom-am3517-experimenter-kit
    • Model Number: SDK-XAM3517-10-256512R
    • (Note: AM3517 EVM will work as well, but full EVM is not required)
  • Ethernet Crossover Cable:
    • (Not included with Zoom Experimenter Kit)
  • USB A to B cable:
    • (Not included with Zoom Experimenter Kit)
    • Note this must be a full-size USB B type connector, not a mini-B connector
  • 2 GB SD Flash card:
    • (Not included with Zoom Experimenter Kit)

Highly Recommended Hardware[edit]

  • Logic PD 4.3” WQVGA Display Kit
    • http://www.logicpd.com/products/display-kits/43-wqvga-display-kit
    • Model Number: LCD-4.3-WQVGA-10R

Optional Hardware[edit]

  • Powered USB Hub
    • Any model should work fine
    • Recommeded that ports support 500mA if webcam will be used
    • Required for USB Keyboard, Mouse and/or Webcam (Experimenter Kit does not provide sufficient current)
  • USB Keyboard
    • Must be connected through a Powered USB Hub
    • Any standard USB keyboard should work fine
  • USB Mouse
    • Must be connected through a Powered USB Hub
    • Any standard USB mouse should work fine
  • USB Webcam
    • Must be connected through a Powered USB Hub
    • USB hub should provide 500mA if used
    • Logitech C200 has been tested and is recommended
      • http://www.logitech.com/index.cfm/notebook_products/webcams/devices/5865&cl=us,en


Software Requirements & Setup[edit]

PC[edit]

  • Windows XP operating system
    • Other operating systems may work, but labs have been tested with Windows XP
  • Windows FTDI driver (CDM 2.06.00 WHQL Certified)
    • Download from workshop repository: http://software-dl.ti.com/trainingTTO/trainingTTO_public_sw/am3517_odw/CDM206000.exe
    • Or from supplier: http://www.ftdichip.com/Drivers/CDM/CDM20600.exe
  • Terra Term Pro
    • Download from workshop repository: http://software-dl.ti.com/trainingTTO/trainingTTO_public_sw/am3517_odw/ttermp23.zip
    • Or from supllier: http://hp.vector.co.jp/authors/VA002416/ttermp23.zip
  • VMWare Player version 3.0 or later
    • Download from workshop repository: http://software-dl.ti.com/trainingTTO/trainingTTO_public_sw/am3517_odw/VMware-player-3.0.1-227600.exe
    • Or from supplier: http://downloads.vmware.com/d/info/desktop_downloads/vmware_player/3_0
    • (Note: this may be replaced with VMware Workstation 6.5.3 or later if available, although this requires a license)
  • Lab Files:
    • Ubuntu-x11-client VMware virtual machine image (10 May 2010):
      • (also contains tar.gz files to image 2 GB SD card)
      • (three part zip file. Download all three files to same folder and open .zip extension)
      • (Your browser may attempt to rename the .z01 and .z02 extensions to zip, don't let it!!!)
      • http://software-dl.ti.com/trainingTTO/trainingTTO_public_sw/am3517_odw/Ubuntu-x11-client3.zip
      • http://software-dl.ti.com/trainingTTO/trainingTTO_public_sw/am3517_odw/Ubuntu-x11-client3.z01 (See warning above)
      • http://software-dl.ti.com/trainingTTO/trainingTTO_public_sw/am3517_odw/Ubuntu-x11-client3.z02 (See warning above)
    • Script files referenced by Lab Setup instructions. (Now this zip also includes the FAE-Summit reflash script.) (5 May 2010)
      • http://software-dl.ti.com/trainingTTO/trainingTTO_public_sw/am3517_odw/am3517_scripts.zip

SD card[edit]

The setup for the target is based upon a 2 GB SD card.
The files found in the Ubuntu-x11-client VMware virtual machine image must be loaded to the card.
Lab Setup directions describe how to image the 2GB SD/MMC card.
The download link for the Ubuntu-x11-client VMware virtual machine is provided above in section Software Requirements - PC.
