Build 3.01.00.06 (Alpha(2) Release)[edit]

Changes from Previous Release

  • Support for C64x de-interlacer.
  • Encode demos integrated with C64x de-interlacer
  • Support for g.711 speech encode in encode demos and g.711 speech decode in decode demos
  • Linux PSP updated from OMAP35x Linux PSP GIT v3.00.00.03 (Linux kernel version 2.6.32-rc5)to AM35x OMAP35x PSP SDK v03.00.00.05 (Linux kernel version 2.6.32)
  • DVSDK Demos, DMAI, DVTB, MFP and DSP Link validated with the AM35x OMAP35x PSP SDK v03.00.00.05

Build 3.01.00.03 (Alpha(1) Release)[edit]

Changes from Previous Release

  • Support for 520MHz DSP and 720 MHz ARM clock
  • Linux PSP updated from OMAP35x Linux PSP GIT v2.01.03 (Linux kernel version 2.6.29-rc3)to AM35x OMAP35x PSP SDK v03.00.00.03 (Linux kernel version 2.6.32-rc5)
  • DVSDK Demos, DMAI, DVTB, MFP and DSP Link validated with the AM35x OMAP35x PSP SDK v03.00.00.03

Refer to the migration guide for more information on moving your applications from GA Patch 2 Release(Build 3.00.02.44) to Alpha 1 Release (Build 3.01.00.03)

Build 3.00.02.44 (General Availability Patch 2)[edit]

Changes from Previous Release

  • Support for OMAP EVM Main Board Rev G.
  • Support for 430MHz DSP and 600 MHz ARM clock
  • Linux PSP updated from OMAP35x Linux PSP GIT v2.01.02 (Linux kernel version 2.6.29-rc3) to OMAP35x Linux PSP GIT v2.01.03 (Linux kernel version 2.6.29-rc3)
  • DVSDK Demos, DMAI, DVTB, MFP and DSP Link validated with the OMAP35x Linux PSP GIT v2.01.03

Refer to the migration guide for more information on moving your applications from GA Patch 1 Release (Build 3.00.01.42) to GA Patch 2 Release(Build 3.00.02.43)

Build 3.00.01.42 (General Availability Patch 1)[edit]

Changes from Previous Release

  • Provided fixes to the DVSDK top level Makefile to perform 'make clobber' and 'make everything'
  • Improved the quality of the Davinci Effect Video streams to perform high quality video decoding
  • Modified the collateral naming conventions
  • The Graphis SDK v3.00.00.09 is made available on the extranet
  • Moved the Graphics SDK Getting Started Guide to online Wiki.

Build 3.00.00.40 (General Availability)[edit]

Changes from Previous Release

  • Linux PSP updated from OMAP35x Linux PSP GIT v2.01.01 (Linux kernel version 2.6.29-rc3) to OMAP35x Linux PSP GIT v2.01.02 (Linux kernel version 2.6.29-rc3)
  • DVSDK Demos and DMAI integrated with the OMAP35x Linux PSP GIT v2.01.02
  • MFP updated from v2.23.01 to v2.24 and validated with OMAP35x Linux PSP GIT v2.01.02
  • DVTB, DSP Link validated with OMAP35x Linux PSP GIT v2.01.02
  • Standalone Codec Server

Refer to the migration guide for more information on moving your applications from Beta 2 (Build 3.00.00.29) Release to GA (Build 3.00.00.40)
