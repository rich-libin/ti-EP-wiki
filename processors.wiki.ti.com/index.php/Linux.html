IRC Channels[edit]

Davinci IRC chat can be found in the #davinci channel on irc.freenode.net. For more information about Freenode, see www.freenode.net

Other channels that might be relevant include #linux-omap, #gst-ti, #beagle, and #rowboat. Non-TI related channels of interest include #ubuntu-arm and #oe.

It is possible to view those channels via a web page by visiting http://webchat.freenode.net/?channels=linux-omap,davinci,beagle,rowboat

General IRC help can be found here :- IRC Help [1]
