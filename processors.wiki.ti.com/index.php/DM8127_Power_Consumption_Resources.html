DM8127 Power Consumption Resources

Introduction
[edit]

The DM8127 is similar to the DM814x family with specific peripherals not used, such as SATA, Camera1A and Camera1B ports. This device has both an ARM8 and DSP core. Power Estimation includes the Fixed supplies, and Core/Processor/(Accelerator if separate) supplies. This discussion tries to illustrate the DM8127 Power Estimation for the DM8127 EVM.

Fixed Supplies

Supply name IO for supply voltage
DVVD IO Standard pins 3.3v, 1.8v
DVVD_GPMC,GPMCB GPMC(B) pins 3.3v, 1.8v
DVVD_RGMII RGMII pins 3.3v, 1.8v
DVVD_SD SD pins 3.3v, 1.8v
DVVD_C C pins 3.3v, 1.8v
DVVD_M M pins 1.8v
DVVD_DDR DDR 2 sets DDR2-1.8v, DDR3-1.5v, DDR3L-1.35v
VDDA_USB 2 USB 1.8v (note special 5v detect is not included)
VDDA_PCie PCIe 1.8v
VDDA_CSI2 CSI2 1.8v
VDDA_HDMI HDMI 1.8v
VDDA_1p8 7 PLL 1.8v, ferrite filters
VDDA_VDAC analog V-DAC 1.8v
VREFSSTL 2 sets-DDR IO Reference .5*DVVD_DDR

Variable power supplies based on OPP mode, see DM8127 datasheet tables 8-4 and 8-5 for values.

Power Supply Type OPP_100 OPP_120 OPP_160
CVVD Logic core (no AVS or dynamic scaling) 1.1 1.2 1.35
CVVD_ARM ARM core (no AVS or dynamic scaling) 1.1 1.2 1.35
CVVD_DSP DSP core (no AVS or dynamic scaling) 1.1 1.2 1.35
CVVD_HDVICP HDVICP core 1.1 1.2 1.35

The operating mode of the DM8127, determines both the speed of the processors and accelerators, and the voltage. Example power dissipation by mode is:

Operating condition SOC Power Estimate (w)
OPP100 2.2w
OPP120 3w
OPP_160 3.6w

Given the DM8127 is a subset of the DM814x, given that CAM1A, CAM1B, and SATA interfaces are disabled, you can follow the DM814x/AM387x example for power estimation[1] .

Example Power Estimation for OPP100 - note you can change parameters, (upper right) OPP mode, and main spreadsheet for customization, is avaiable here. File:DM814x AM387x PowerSpreadSheet RevE0.zip

Customization can further lower the estimate. You need to enter the thermal parameter in the upper right. But a thermal calculation, including the heatsink, conduction to case, airflow is not included in this spreadsheet.
