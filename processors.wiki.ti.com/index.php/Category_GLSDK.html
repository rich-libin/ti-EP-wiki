Processor SDK Linux Automotive Landing page [edit]

This page provides information about Processor SDK Linux Automotive releases for Automotive In Vehicle Infotainment.

NOTE: Starting from kernel version 4.4, GLSDK is renamed as "Processor SDK Linux Automotive" and versioning was also reset from 7.xx to 3.00. The major version is incremented for every LTS Linux Kernel version. The major release 3.xx was on K4.4 (2016 LTS). There was no Processor SDK Linux Automotive release on K4.9 (2017 LTS). The release 5.x was on K4.14(2018 LTS). The current major release 6.xx is based on K4.19 (2019 LTS).


Last updated: 12/16/2019
