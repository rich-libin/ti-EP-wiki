Sitara SDK 4.01 Release Notes
[edit]

Release Notes - Jan 21, 2011


Introduction[edit]

This is Release 4.01 of the Sitara Software Development Kit (SDK). The SDK serves to provide a fundamental software platform for development, deployment and execution of Linux based applications on the following ARM Microprocessors:

  • AM1810
  • AM180x
  • AM35x
  • AM37x


Component Version Base Repository Base Tag (or commit id)
OMAP Linux Kernel 2.6.32(Linux kernel version) The baseline was chosen from "pm" branch of http://git.kernel.org/?p=linux/kernel/git/tmlind/linux-omap-2.6.git;a=shortlog;h=refs/heads/pm 3fd5969a81a9324b58a1c19cf510c0da97c99565. For detailed information on the various software components included in the release and how to use them, please refer to the Software Developer's Guide.

The kernel and U-Boot have been compiled with CodeSourcery GNU Toolchain for ARM Processors 2009-q1-203 based on GCC 4.3.3

Documentation[edit]

The Software Developer's Guide provides instructions on how to use SDK 4.01. A copy of the Software Developer's Guide is included in the docs folder of the SDK installation.


What's Supported[edit]

Software components included in this SDK are::

  • PSP versions
    • 0x.0x.0x.0x - AM1810x
    • 0x.0x.0x.0x - AM180x
    • 03.00.01.06 - AM35x/Am37x
  • SDK Installer
  • Setup Scripts
  • Makefiles
  • Matrix Application Launchers
  • Example Applications & Demos
  • Code Composer Studio v5
  • Host Tools
    • Flash Tool
    • Pin Mux Utility



What's Not Supported[edit]

The following items are not supported:

  • AM35x support in the provided Flash Tool. Future releases which will include this support are located here:
  • For details of unsupported features with WLAN and BT Components see associated Release Notes

New in this Release[edit]

  • New platforms including AM1810, AM180x and Am35x
  • New Matrix enhancements and updates
  • New release of CCSv5
  • New Remote System Explorer plugin for CCSv5
  • New Setup Script for FS, TFTP, NFS, Minicom and Uboot configurations
  • Updated SDK Installer

Please refer to the Software Developer's Guide for more information on how to install the Sitara SDK.
