H/W designer's resource[edit]

  1. Make sure you have the latest version of the DM648/7 datasheet.
  2. Refer to DM648 EVM H/W design, made by Lyrtech.
  3. Go through DM648 silicon errata and know the latest silicon revision, silicon bug and the workaround.
  4. Know the power consumption of DM648/7.
  5. Thermal Considerations For the DM64xx, DM64x, and C6000 Devices (spraal9.pdf)
  6. PCB design:
    • High Speed DSP Systems Design Reference Guide (spru889.pdf)
    • Understanding TI's PCB Routing Rule-Based DDR Timing Specification (spraav0.pdf)
    • Implementing DDR2 PCB Layout on the TMS320DM644x DMSoC.
    • C6457 SerDes Implementation Guidelines (for SGMII)
    • You can download the simulation models from the product folders, DM648.
  7. TMS320C6000 DSP Designing for JTAG Emulation Reference Guide.
  8. Hardware Design Checklist
  9. TMS320DM648/7 SoC Architecture and Throughput Overview.

S/W(including driver) designer's resource[edit]

  1. Here is DVEVM and DVSDK overview.
  2. Register your DM648 EVM here by the S/N of your EVM. Then you will be authorized to download the latest DVSDK, PSP driver and Ittiam demo via same website. 
  3. For DM648/7, TI has no CSL. You can use DM648/7 PSP driver. Download the latest PSP driver by your my.ti.com account.
  4. Using a PSP Driver in DSP/BIOS.
  5. Network Developers Kit FAQ.
  6. Request Evaluation Codec From TI.
  7. Quickly Getting Started on TI Codec Engine.
  8. C6000 cgtools and Optimization Techniques for the TI C6000 Compiler. 
  9. Request TI Video Vision Library.
  10. C64x+ Image/Video Library.
  11. C64x+ IQMath Library - A Virtual Floating Point Engine.
  12. C64x+ Math Library.
  13. VICP Signal Processing Library.
  14. How to operate EDMA3 of DM648/7?
    • Use TI EDMA3 LLD low level driver based on DSP/BIOS. Here is an example How to use the EDMA3 Driver on a TMS320DM643x Device.
    • Programming EDMA without EDMA3LLD package.
    • Use TI Framework Component based on DSP/BIOS.
    • Write own CSL by referring to EDMA3 CSL of C6455.

Remark[edit]

  1. You can download TI S/W components DSP/BIOS, NDK, XDAIS and so on.
  2. You can download TI CodeGenerationTools via your my.ti.com account.
