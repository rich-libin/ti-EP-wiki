General Description[edit]

The AM35x SOM-M2 integrates the Texas Instruments Incorporated (TI) Sitara™ AM35x microprocessor and the WiLink™ 6.0 wireless chipset that contains multiple radio technologies. At the heart of WiLink 6.0 sits the WL1271 device, which encompasses all digital and analog content required for WLAN and Bluetooth® communication, and enables support for IEEE 802.11b/g/n and Bluetooth 2.1 + EDR. This FCC-certified System On Module (SOM) is a production ready hardware and software solution that fast forwards embedded designs while reducing risk and controlling costs.

The compact size of the AM35x SOM-M2 is ideal for medical patient monitoring wearables and other portable instrumentation applications. For commercial signage, medical imaging, avionics, and industrial displays, the AM35x SOM‑M2 allows for powerful versatility and long-life.

FCC-Certified AM35x SOM-M2 Highlights:

  • Complete out-of-the-box system gets customers connected and into production quickly.
  • Pre-certification saves customers significant time and money by reducing design risk and shortening the final electromagnetic compatibility (EMC) testing phase.
  • U.FL connector on the SOM printed circuit board (PCB) enables flexible antenna architecture allowing customers to design the AM35x SOM-M2 into a multitude of end applications.


For more information about the AM35x SOM-M2 and to learn where to buy it, visit Logic PD's web-site: http://www.logicpd.com/products/system-modules/texas-instruments-am35x-som-m2

Product Features[edit]

Processor

  • TI AM3517 processor with ARM Cortex-A8 core running at 600 MHz

SDRAM Memory

  • DDR2 256 MB (scalable to 512 MB)

Flash Memory

  • NAND flash 512 MB (scalable)

Display & Graphics

  • SGX530 graphics accelerator (AM3517 only)
  • Programmable color LCD controller supports up to a 24 bpp TFT interface

Touchscreen

  • Integrated 4-wire touchscreen controller

Network Support

  • 10/100 Base-T Ethernet PHY*
  • 802.11b/g/n wireless Ethernet
  • Bluetooth 2.1 + EDR PC Card Expansion

PC Card Expansion

  • Multiple SD/MMC card support

USB

  • One USB 2.0 high-speed On-the-Go interface
  • One USB 2.0 high-speed host interface

Serial Ports

  • Up to four external UARTs
  • Three I2C ports
  • Three SPI ports

GPIO

  • Programmable I/O depending on peripheral requirements

Software

  • U-Boot (bootloader/monitor)
  • Open source Linux BSP

Mechanical

  • SOM-M2 form factor
  • 40.9 mm wide x 51.2 mm long x 5.4 mm high

FCC Part 15 Subpart C Compliant

RoHS Compliant

Software[edit]

Release 03.00.00.05 of OMAP PSP Software Development Kit (SDK) serves to provide a fundamental software platform for development, deployment, and execution of Linux based applications on Texas Instruments AM3517 EVM Development Kits. The User Guide and download information is available here.

The WiLink 6.0 patch set supports WLAN and Bluetooth applications using the Zoom™ AM3517 EVM and the AM3517 PSP from Texas Instruments (TI). The WiLink software patch set is applied to TI’s open source Linux PSP version 03.00.00.05 and includes a pre-built sample image to quickly demonstrate connectivity as well as complete instructions for building a custom image from scratch. The WiLink 6.0 patch set is available from the Logic PD website here.
Note: Registration of an AM3517 EVM Development Kit is required to access the WiLink 6.0 patch set.

Development Kits[edit]

Developed with Logic PD, the AM3517 Evaluation Module (EVM) enables developers to immediately start evaluating AM35x processors and begin building industrial applications such as single board computers, data terminals, point of sale/service, digital signage and numerous other products requiring high performance.

Technical information and schematics are available here.

AM3517 EVM Development Kit is available through the TI eStore and distributors. AM3517 Evaluation Module
