Overview[edit]

The CCSv5 Fundamentals Workshops are roughly half-day workshops for beginners who wish to learn the basics of the CCSv5 environment. The content covered varies depending on the device.


MSP430  [edit]

Software requirements 
  • CCSv5.3 or later 
  • MSP430ware (included in CCS)
  • Labs (see 'Materials' sections)
Hardware requirements  MSP430G2231 or MSP430G2553 LaunchPad 
Target Audience  New users, or users new to CCSv5
Length  3-4 hours
Modules Included 
  • Overview
  • Getting Started
  • Watchpoints
  • Build properties
  • Debugging programs existing in flash
  • Portable Projects
  • ULP Advisor
  • GRACE basics
Materials
  • Presentation 
  • Lab Presentation 
  • Lab 1 Video run through 
  • Lab 2 Video run through 
  • Lab 3 Video run through 
  • Grace Lab Video run through 
  • Lab: The presentation assumes the lab zip file is extracted in the root C: drive

C2000  [edit]

Software requirements
  • CCSv5.5 
  • controlSUITE: Make sure to install F2806x support.
Hardware requirements  F28069 Piccolo Experimenter Kit
Target Audience  New users, or users new to CCSv5
Length  3-4 hours
Modules Included 
  • CCS Overview 
  • Eclipse environment concepts 
  • Creating projects 
  • Watchpoints (data breakpoints) 
  • Debugging existing code in flash 
  • Profile clock 
  • Project options 
  • Target Configurations
  • Portable Projects
  • Real-time Mode 
  • UniFlash
Materials
  • Presentation 
  • Labs
  • 中文 (OLD version)

AM3359 (BeagleBone)[edit]

Software requirements
  • CCSv5.2.1
  • STARTERWARE-AM335X version 2.00.00.06
  • TMS470 Compiler release 4.9.6 installed under the directory C:\ti\TMS470 4.9.6 Compiler
    • Available at the CCS Update site page.
  • Eclipse terminal plug-in (installation details shown in the materials section below)
Hardware requirements BeagleBone
Target Audience New users, or users new to CCSv5
Length 2~3 hours (can go longer depending on the questions)
Modules Included
  • CCS Overview
  • Eclipse environment concepts
  • Creating projects
  • Watchpoints (data breakpoints)
  • Profile clock
  • Project options
  • Target Configurations
  • Optimization
  • Portable Projects
Materials
  • 3-hour version presentation
  • 2-hour version presentation
  • Labs
  • Detailed installation instructions
  • Simple cheat sheet for easy copy and paste on the labs

Software requirements
  • CCSv5.3 or later
  • StellarisWare
  • PuTTY (download details shown in the materials section below)

Target Audience New users, or users new to CCSv5
Length 2~3 hours
Modules Included

  • Eclipse environment concepts
  • Creating projects
  • Watchpoints (data breakpoints)
  • Profiling clock Cycles
  • Project options
