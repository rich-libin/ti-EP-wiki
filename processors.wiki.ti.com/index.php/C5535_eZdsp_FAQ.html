How to meaure Power Consumption with the C5535 eZdsp?[edit]

Following jumpers can be used for power consumption measurement

- JP1: CPU Core (CVDD) 

 1) the R20 should be removed first.
 2) connect current meter between JP1 pin 1 and 2. 
 *CVDD is supplied by the DSP_LDO as default (1.3V) with R124 (zero ohm)

- JP4: IO1 (DVDDIO) 

 1) the R97 should be removed first. 
 2) connect current meter between JP4 pin 1 and 2.

- JP5: USB 3.3V (USB_VDDOSC + USB_VDDA3P3 + USB_VDDPLLDVDDIO) 

 1) the R96 should be removed first. 
 2) connect current meter between JP5 pin 1 and 2.

- JP2: Total board power 

 1) Use J1 for USB Connection
 2) connect current meter between JP2 pin 1 and 2.
