[资源简介] About this resource guide[edit]

There are plenty of resource in TI website and here just make it clearer for you to get the entrance rapidly without much documentation first. You can go thru these resource link and get the concept for the chip / platform quickly.

For deeply development, you should read the specs, Application notes and resource in AM35x Wiki carefully. For any question / issue, you can leave your message to TI E2E™ Community - Sitara™ ARM® Microprocessors.

[硬件设计资源] H/W designer's resource[edit]

TI provides a rich set of documents for a good hardware design.

  1. [评估板]TI provides the verified reference boards for your development that is EVM. For AM35x, there are two EVMs:
One is from LogicPD. Please refer to AM35x EVM H/W design, you need to register LogicPD for further downloading.
The other is from Mistral, the Craneboard. It's a 4-layers low cost design, with schematic, gerber file, and Linux BSP provide.
  1. [芯片资料]Go to AM35x chip documents for the latest chip User Guides(TRM), Data sheet (DM), and Silicon Errata(Silicon bug and the workaround).
  2. [原理图设计]You can get the schematic check list and Hardware Design Checklist for your reference.
  3. [PCB设计]PCB design:
    • AM35x Via Channel Array PCB layout and the example in Allegro
    • Please check AM35x TRM 6.4.2 SDRAM Controller (SDRC) for mDDR/DDR2 layout Guidelines
    • USB 2.0 Board Design and Layout Guidelines
    • Refer to appendix C of nFBGA Packaging application note for thermal modeling results
    • OrCAD, Allegro封装
    • [BSDL,IBIS仿真模型]You can get the Simulation Models for AM35x here.
  4. [电源设计]Power design:
    • Analog and Power for AM35x
    • AM35x with TPS65023: Design In Guide
    • AM35x with TPS650732: Design In Guide
  5. [AM35x功耗评估工具]AM35x Power Estimation Spreadsheet
  6. [OMAP35x到AM35x的移植参考]OMAP35x to AM35x Hardware Migration Guide
  7. [无线连接功能]ARM Processor Wireless Connectivity.
  8. [SD Card]How to design with the SD-MMC interface on the AM35x processors. Please refer to the AM35x SD-MMC Usage Notes.

[软件设计资源] S/W designer's resource[edit]

TI has rich reference software for customer development including Linux, Android, WINCE and BSL.

  • [Linux]TI has developed Linux PSP 03.00.xx.xx for AM35x(PSP means Platform Support Products). With each PSP release, there are Release Notes, UserGuide, and Feature Performance Guide. Besides these, there are plenty of application notes, such as: how to setup the environment, toolchain setup, etc. Please find all these from Sitara Linux.

    Please download Linux EZSDK source code in below website: Linux for AM35X downloads.    

    You can find more info like Graphics SDK from AM3517_Product_Downloads.

  • [Android]Based on PSP 03.00.xx.xx for AM35x, there is TI_Android_GingerBread_2_3_4_DevKit_2_1 .

    It's the snapshot of TI rowboat project, with full kerenel source code, testing, release notes and userguide.

  • [Windows CE7]TI Windows Embedded Compact (WEC) 7.0 Software Development Kit (SDK) is now available. Download the WEC 7.0 SDK  from here.
  • [BSL]Board support libs - which is focusing on the Board components verification. For AM35x EVM, please find at LogicPD BSL.
  • [QT]QT is a cross-platform applications and UI development framework, how to AM35x established on the QT.
  • [Adobe Flash]You can download from here.It is only for TI Cortex-A8.
  • [Encryption software] This software enables encryption, crypto for AndroidTM operating system. And Encryption softwares include OpenSSL, Dropbear SSH, Linux Kernel Crypto and Raccoon.
  • [AM35x引导手册]This Getting Started Guide (GSG) walks you through setting up a development system for the AM35x.

[工具链] Tools & Utilities[edit]

Maybe some additional utilities are needed for downloading, compiling, and development.

  • [Linux及Android工具链]First to download ubuntu here : Ubuntu SW. The installation methods please refer to the link.

    For more detail info, please refer to the below website:Installing the Toolchain,and download the toolchain from here.

  • [WINCE工具链]The following software must be installed prior to installing the AM35x BSP. Note that the software must be installed in the order listed:
    • Visual Studio 2005 (VS2005)
    • Visual Studio SP1 (Service Pack 1)
    • Windows Embedded CE 6.0 (Plug In for Visual Studio 2005)
    • Windows CE 6.0 SP1 (Service Pack 1)
    • Windows CE 6.0 R2 Update
    • Windows CE 6.0 Cumulative Product Update Rollup Package 2007 (QFEs)
    • Remaining Windows CE 6.0 QFEs
    • You can download most of these tools except Visual Studio 2005(could not be downloaded online) at below website: TOOL Download
  • [ROM CODE下载工具]You can download the Flash Loading Utility for AM35x here.
  • [CCS IDE]You can get Sitara support from here.
  • [AM35x DDR register calc tool]The link describes how to set the SDRC registers in the AM35x SDRC module depending on the mDDR or DDR2 memory that you are using.
  • [Pin Mux Utility]The Pin Mux Utility for Sitara ARM Cortex-A8 based microprocessors is a Windows-based software tool for configuring pin multiplexing settings and I/O cell characteristics.You can refer to this link. 
  • [DSP&ARM MPU 选择工具]You can find the DSP&ARM MPU choose Tool, refer to DSP & ARM MPU Selection Tool.
