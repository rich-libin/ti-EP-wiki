Overview[edit]

The CCS Fundamentals Workshops are roughly half-day workshops for beginners who wish to learn the basics of the CCS environment. The content covered varies depending on the device.


CCSv6[edit]

MSP432[edit]

MSP432P4x[edit]

Software requirements 
  • CCSv6.1.3 
    • Note that CCSv6.2.0 introduces a Resource Explorer v2.0. This workshop was created with Resource Explorer v1.x. If you are using CCSv6.2.0, the older Resource Explorer is still available under the 'View' menu as "Resource Explorer Classic"
  • MSPWare 3.30 or later (Download via CCS App Center or from ti.com)
Hardware requirements  MSP432P401R LaunchPad
Target Audience  New users, or users new to CCSv6
Length  3-4 hours
Modules Included 
  • Overview
  • Getting Started
  • Basic Debugging
  • Optimizer Assistant
  • Real-time Memory Debug
  • SWO Trace
Materials
  • Main Presentation 
  • Lab Handbook
