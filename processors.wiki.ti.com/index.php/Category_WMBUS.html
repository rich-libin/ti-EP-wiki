Wireless MBUS[edit]

Back to sub-1 GHz page

Welcome to the Wireless MBUS wiki! At this page we have collected everything concerning the wireless MBUS standard, such as radio register settings, code examples and links to useful information.

Supported Hardware Platforms[edit]

  • EXP430F5438 + CC1120EM
  • EXP430FG461x + CC1101EM / CC110LEM
  • CC430 EVM
  • CC1200DK
  • CC11xLDK
  • CC1120+ SKY 169MHz
