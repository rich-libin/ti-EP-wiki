What is your project?[edit]

Project name[edit]

Visual Traffic Device

Project Description[edit]

  1. What are you making? A device similar to a current GPS device but bases route on traffic signals
  2. For whom are you making it, and why do they need it? Everyday consumers would need/want this to save time while driving
  3. What technologies (programming languages, etc.) will you be using? Android + Java
  4. In general, feel free to add as much content as you think is valuable. Consider this template as the bare minimum.Touchscreen for ease of interaction and appeal

Project Scope[edit]

We are attempting to complete this project in one semester. Having a class that allows us to work on open projects gives us time within school to attempt to complete this. Anthony has had a lot of previous programming experience and is willing to learn about the Android java system. Chris is more hardware focused and is interested in circuits and how hardware interacts with each other.

Our project is separated into several parts: Hardware: Beagleboard�+LCD and Expansion Board Software: Database�+Travel Algorithm+Android GUI
