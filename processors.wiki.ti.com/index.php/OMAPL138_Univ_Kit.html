Software & Documentation for the TMDSEXPL138-UNV OMAP-L138/C6748 DSP Experimenter Kit for Universities

First, download the EVM documentation and utilities in the EVM Downloads section to get familiar with the hardware and sample code.

Next, download important software, such as Code Composer Studio to begin development. For Linux or DSP development, pick up the additional software components provided as well.

Finally, for documentation and support on the OMAP-L138 chip, see the Support and Documentation section.

Downloads[edit]

EVM Utilities and Documentation[edit]

  • EVM User Guide - This document gives an overview of the EVM and the various components on the board. It also details how to use CCS to run the example code provided below.
  • CCS Board Support Package (BSP) & GEL Files - The BSP contains CCS examples for using most of the peripherals on the device. GEL files are also included which will configure the mDDR memory and the PLLs.
  • EVM Hardware Design Files - This package contains the hardware schematics for the SOM and baseboard, which can be useful for debugging.
  • Serial Flashing Utility - This utility will allow you to program the SPI Flash on the EVM using the serial port of the EVM, without the need for an emulator or CCS.
  • Getting Started Guide - The Getting Started Guide (GSG) will walk you through the steps necessary to get Linux up and running on the EVM.

Software[edit]

  • Code Composer Studio - Code Composer Studio is the IDE used for developing and debugging applications running on the processor. Educators should contact the TI University Program for free access to Code Composer Studio for projects and labs.
  • StarterWare for OMAP-L138 - StarterWare provides C-based no-OS platform support for the ARM and DSP found on the OMAP-L138.
  • DSPLIB for C6748 DSP - DSPLIB is a platform optimized DSP function library which can be used for developing DSP applications.
  • Davinci Platform Support Package (PSP) - The PSP contains the source code and pre-built binaries for the Linux kernel and U-boot. In addition, it contains board level tools for programming the SPI Flash in CCS, along with pre-built UBLs (secondary bootloader) needed to boot U-boot.
  • Linux Software Development Kit (SDK) for OMAP-L138 - The SDK contains the Davinci PSP, which can be downloaded separately above, along with additional packages useful for Linux development. This is a binary install file which can only be run on a Linux host.

Support and Documentation[edit]

  • OMAP-L138 Product Folder - The product folder for OMAP-L138 contains documentation for the chip along with application notes and additional information. The most useful documents are linked below:
    • OMAP-L138 Datasheet
    • OMAP-L138 Silicon Errata
    • OMAP-L138 Technical Reference Manual
  • OMAP-L138 Design Companion Wiki - This design companion wiki contains links to additional support documentation and examples not found in the device datasheet or application notes.
  • OMAP-L138 Support Forum - For any questions related to the OMAP-L138 hardware, search the forum or create a new post.
