Overview[edit]


This topic indicates the trace features supported by device and provides links to instructions on how to setup each feature.


Features[edit]

Feature Description
ETB - ARM ARM CPU trace directed to the onchip Embedded Trace Buffer
ETB - DSP DSP CPU trace directed to the onchip Embedded Trace Buffer
ETB - STM System trace directed to the onchip Embedded Trace Buffer
STM System trace exported via XDS560v2 class emulator
ARM - PIN ARM CPU trace exported to pins and captured by a trace receiver. Often called ETM trace.
DSP - PIN DSP CPU trace exported to pins and captured by a trace receiver such as XDS560T (obsolete) or XDS560v2 Pro Trace
ITM Instrumentation Trace Macrocell.   Also called 4pin TPIU.  ITM is printf() style debugging to trace operating system and application events.
SWO Serial Wire Output.  Also called 2pin TPIU.
PTM Program Trace Macrocell.  Provide trace of program flow.
