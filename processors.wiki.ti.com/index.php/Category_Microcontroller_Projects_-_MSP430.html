Microcontroller Projects for the MSP430[edit]

Please check out our new Microcontroller Projects page!

This is a collection of open source projects for the MSP430 microcontroller. Each project includes all the source code and schematics for anyone to be able to reproduce the application. If you've developed an MSP430 project and would like to share your work, please contribute it here.
