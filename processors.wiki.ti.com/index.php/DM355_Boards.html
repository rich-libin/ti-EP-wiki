Leopard Board[edit]

The Leopard Board is a full featured, ultra low cost, small form factor, high performance development system which includes TMS320DM355 Processor board and a VGA camera board to provide VGA resolution video capture. Read more about LeopardBoard on: leopardboard.org , eLinux wiki , RidgeRun free SDK .
see also: Leopard Board DM355 Overview
For more information on how to get started with the Leopard Board: Getting Started Guide for Leopard Board DM355

Z3 DM355 Boards[edit]

There is a module shaped board Z3-DM355-MOD that can be either plugged into the Z3 application backplane Z3-DM355-APP or into a custom design.

Z3 tells about the application backplane:

Z3 's low-power DM355 multimedia application board is used in conjunction with our Z3-DM355-MOD low power module based upon Texas Instruments DM355 DaVinci series processor. Our Z3-DM355-APP application board is 100mm x130mm in size.
Z3 offers open-source Linux or Monte Vista Linux in addition to multi-media framework based upon open-source Gstreamer. The module contains options for additional audio codecs, image codecs and application software.
Applications for the Z3-DM355-APP-01 available but not limited to are: IP Camera, photoframe, medical and industrial video capture/playback applications. - Source: HD DM355 Application

APPRO DM355 IP Networking Camera[edit]

This is a web cam hardware design including a very sensitive and flexible 5 mega pixel imaging sensor from Aptina. It served as an early access platform for many TI customers.

sprt459f had this headline for it:

TMS320DM355 IP Camera Reference Design: DM355IPNC-MT5 @ U.S. $795 - Source: sprt459f - HD Video Surveillance IP Camera Reference Designs (PDF)

VCT IP CAM[edit]

This design is based upon the APPRO IP CAM. It further includes a signal splitter for the digital sensor data so that a second processor of Type DM6435 could be integrated. The determined purpose of that add on board is high performance video analytics or an alternate encoding format whilst the original DM355 will mainly can concentrate on the built in video compression duty. As both devices are programmable its possible to adapt this standard role assignment in a wide variety.

sprt459f had this headline for it:

Video Content Analysis (VCA) DM355 IP Camera Reference Design: DM355IPNC-VCA1 @ U.S. $995 - Source: sprt459f - HD Video Surveillance IP Camera Reference Designs (PDF)
