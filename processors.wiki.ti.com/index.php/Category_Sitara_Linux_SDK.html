Thanks for your interest in the Sitara Linux SDK. This page stores links to all the pages relevant to the SDK. There's quite a bit of documentation available, and it might seem a bit overwhelming. The place to truly get started is the Getting Started Guide. There is also a "portal" of information called the Sitara Linux Software Developer’s Guide. Both of these resources attempt to summarize all of the resources available to Linux developers using Sitara ARM products.

Essentially, the SDK is a Linux Distribution with examples, documentation, and tooling to help developer's get moving quickly. It has four major components that you might want to explore in detail depending on your needs:

  • U-Boot Bootloader
  • Linux Kernel
  • Filesystem Featuring Matrix GUI and Example Applications
  • Tools for Developing and Deploying
