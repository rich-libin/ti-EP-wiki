How to run Signal Analyzer demo?[edit]

Refer to "Running the Integra demo examples" in the Software Development Guide on how to run integra demos & Choose the signal Analyzer demo from the choices Available

GUI[edit]

Below is the signal analyzer demo that is seen on the display

Features supported by the Demo[edit]

This Demo supports below signal type 

 - Sine wave
 - Any input signal feed from file (16bit - single channel)

Supports below type of analysis 

 - Amplitude spectrum
 - Power spectrum

User control operations 

 - Zoom
 - Selection for Frequency, sampling frequency, Amplitude & Phase
 - Terminate the demo

Selection of different Signal types[edit]

The drop down box is available in the GUI to select the any of the below input signal type 

 Sine Wave - The sine wave is generated at runtime using other user selected parameters.
 Audio Wave - The default audio file fft_demo_8khz.pcm is choosen for playback, and this parameter is set.
 Input Signal - The file browser dialog window opens up to choose the input file. The file format must be PCM (16 bit, single channel)

Control on the signal parameters[edit]

For parameters can be controlled by the user using mouse, The slide bar, spin box available in the GUI to control these parameters 

 - Frequency (Only for Sine wave signal type)
 - Sampling frequency (Required for audio/input signal type to plot them correctly on the screen)
 - Amplitude (Only for Sine wave signal type)
 - Phase (Only for sine wave signal type)

Zoom[edit]

The button named "Zoom" present in the GUI, to control this, once left click on this button, move the cursor to graph area and group the area to zoom by click and drag, This observes the zooming functionality.

Zoom out can be performed in two ways 1. By right click, 2. By choosing reset button.

Reset[edit]

After zoom operation, press this button to go back to normal state

Analyze/resume[edit]

This button is left for Future scope, currently this button changed to resume, Press this button to go back to normal playback.

Exit the demo[edit]

Click on the exit button displayed on the GUI to terminate the demo
